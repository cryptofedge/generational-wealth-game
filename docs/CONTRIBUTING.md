# Contributing to the Generational Wealth Game

Thank you for your interest in contributing! This project grows stronger with every educator, practitioner, and learner who shares their knowledge.

---

## How to Contribute

### 🐛 Reporting Issues
If you find incorrect information, broken links, or unclear explanations, open a GitHub Issue with:
- A clear title describing the problem
- Description of what's wrong
- Suggested fix (if you have one)

### ✍️ Adding Lessons or Resources
1. Fork the repository
2. Create a new branch: `git checkout -b feature/lesson-name`
3. Add your content to the appropriate folder:
   - `lessons/` — numbered educational modules
   - `resources/` — guides, reading lists, references
   - `tools/` — calculators, worksheets, planning aids
   - `docs/` — deep-dive documentation and case studies
4. Follow the existing file naming convention (e.g., `03-topic-name.md`)
5. Submit a Pull Request with a clear description of what you added and why

### 📐 Content Standards
- Write in **plain language** — this is for everyday people, not finance experts
- Always include a **Disclaimer** section if providing numbers or financial projections
- Cite your sources where possible
- No promotional content, affiliate links, or product endorsements
- Keep it factual and educational

---

## File Naming Convention

```
lessons/01-topic-name.md
lessons/02-next-topic.md
resources/topic-name.md
tools/tool-name.md
docs/document-name.md
```

---

## Commit Message Style

Use clear, descriptive commit messages:
- `add: lesson 03 — how dividend-paying policies work`
- `fix: correct interest rate example in lesson 01`
- `update: expand glossary in recommended-reading.md`

---

## Code of Conduct

Be kind, be accurate, be helpful. We're all here to learn and build generational wealth for our families. Disrespectful, promotional, or misleading contributions will not be merged.
